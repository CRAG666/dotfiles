# Env Dev

## Quick start

This repository is built with Makefile. It is an attempt to set up a minimal environment for development and entertainment on arch linux.

### List of commands

```bash
make help
```

### For P53

```bash
make p53
make scroll
make dns
```

### L14

```bash
make l14
make scroll
make dns
```
