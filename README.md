# Env Dev

## Quick start
This repository is built with Makefile. This is intended to configure a minimal development environment in archlinux

### List of commands
```` bash
make help
````

### Links to only dotfile
```` bash
make init
````

### Vscode costomize
```` bash
make code
````
### Install all

```` bash
make allinstall
````
